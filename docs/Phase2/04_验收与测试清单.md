# Phase 2 验收与测试清单

## 验收标准

1. 可通过 API 创建/查询/删除多个知识库元数据  
2. 每个知识库有独立目录与索引状态，互不干扰  
3. 入库任务可查询状态，失败有错误信息  
4. `/api/chat` 在指定 `kb_id` 下可返回对应知识库结果  
5. 现有 `agv_demo` 兼容不回退（Phase 1 功能不破坏）
