# Phase 3 阶段计划

## 阶段概述

Phase 3 的核心目标是将已完成的后端能力（Phase 1 RAG 流程 + Phase 2 知识库管理）包装为可用的 **H5 响应式界面**，供内网用户在 **PC 与手机浏览器** 中直接使用，无需安装任何客户端。

## 开发约束

- **无构建依赖**：不使用 Webpack/Vite，所有 JS/CSS 通过 CDN 或本地 vendor 文件引入
- **内网部署**：CDN 资源需下载到本地 `frontend/vendor/` 目录，避免访问外网
- **单后端服务**：Flask 同时提供 API 和静态文件服务，无需 Nginx
- **浏览器兼容**：Chrome 90+、Edge 90+（内网环境可控）
- **响应式（新增）**：同一套 H5 在 **桌面与手机浏览器** 上均需可用；对话为主路径 **手机必达**，管理后台以 **可读、可完成核心操作为准**（允许布局简化为纵向/抽屉，不要求与桌面像素级一致）

## 里程碑拆分

### Sprint 1：基础框架 + 对话页（约 3 天）

**目标**：可以打开页面、选择知识库、发送消息并看到流式回答

- [ ] 搭建 `frontend/` 目录结构，下载 vendor 依赖
- [ ] Flask 注册静态文件路由（`/`、`/admin`、`/login`）
- [ ] 实现知识库选择器（**对话页左侧边栏**，调 `GET /api/kb`）
- [ ] 实现消息输入 + 发送（调 `POST /api/chat`）
- [ ] 实现 SSE 流式接收，逐字渲染到对话气泡
- [ ] 实现 Markdown 渲染（表格、代码块、列表）
- [ ] `viewport` 与基础响应式：窄屏下对话侧栏以**抽屉/遮罩**呈现，主区与 Composer 不挤压溢出

**验收**：能完整走通一次 AGV 换电操作问答，并看到引用来源；**视窗宽度约 390px（或团队约定断点）下可完成同路径**

---

### Sprint 2：来源面板 + 图片展示（约 2 天）

**目标**：每次回答附带可展开的"引用来源"卡片，包含文档片段和关联图片

- [ ] 解析响应 JSON 中的 `sources` 字段
- [ ] 渲染来源卡片（文档标题 + 段落摘要 + 图片缩略图）
- [ ] 图片点击放大（Lightbox 效果）
- [ ] 多来源折叠展示（默认折叠，点击展开）

**验收**：来源卡片正确展示图片，图片可点击放大

---

### Sprint 3：管理后台（约 4 天）

**目标**：管理员可以在界面上完成知识库全生命周期管理

- [ ] 知识库列表页：状态徽章、文档数量、操作按钮
- [ ] 新建知识库表单（名称、类型、描述）
- [ ] 知识库详情页：文档列表 + 文件上传
- [ ] 文件上传组件（拖拽 + 点击，带进度条）
- [ ] 索引构建状态实时刷新（轮询 `/api/kb/<id>/status`）
- [ ] 删除知识库 / 删除文档（确认对话框）

**验收**：可完整演示"新建知识库 → 上传文档 → 等待索引完成 → 切换到对话页问答"全流程

---

### Sprint 4：体验打磨（约 2 天）

**目标**：补全错误处理、加载状态、空状态，达到生产可用水准

- [ ] 全局 Toast 通知（成功 / 失败 / 警告）
- [ ] 加载骨架屏（知识库列表、文档列表）
- [ ] 空状态插画（没有知识库、没有文档、没有对话历史）
- [ ] 网络错误自动重连提示
- [ ] **手机端体验收尾**：安全区（`safe-area-inset-*`）、触控热区（建议 ≥44px）、横滑表格/卡片切换策略、管理页小屏 Modal 不溢出
- [ ] 登录页壳（Phase 4 接入前显示"内网免登录"提示）

**验收**：断网、服务异常时有友好提示，不白屏崩溃；**主流手机竖屏**下对话与管理核心流程可完成（见 `03` §0、`02` 响应式章节）

---

| Sprint 4 | 体验打磨（含手机端收尾） | 2～3 | 11～12 |

## 验收标准（Phase 3 完成条件）

1. 对话页可正常选择知识库、提问、流式接收回答
2. 来源卡片正确展示引用文档和图片
3. 管理后台可完成知识库 CRUD + 文档上传 + 索引状态查看
4. 所有页面在 Chrome 90+ 下无 JS 报错
5. 后端无 5xx 错误时，前端不出现空白页
6. **响应式**：在团队约定的窄屏宽度（建议以 **375～390px** 为自测基准）下，对话页可完成选库、提问、流式阅读、展开来源与查看大图；管理页可完成列表浏览、进入详情、上传与触发索引（布局可为简化版）
